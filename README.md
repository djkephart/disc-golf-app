# 🥏 DiscGolf App

A disc golf companion app built with SvelteKit and Tailwind CSS. This is an ongoing portfolio project showcasing modern frontend development.

## Live Features

- **Course Browser** — Browse disc golf courses with details like holes, par, difficulty, and amenities
- **Course Detail Pages** — View in-depth info for each course
- **Scorecard Wizard** — Log a round hole by hole with live score tracking and par/birdie/bogey indicators
- **Dark Mode UI** — Clean, modern dark theme throughout

## Tech Stack

- [SvelteKit](https://kit.svelte.dev/) — Frontend framework
- [Tailwind CSS](https://tailwindcss.com/) — Styling
- [Vite](https://vitejs.dev/) — Build tool

## Planned Features

- 🗺️ Interactive course map
- 👤 Player profiles and round history
- 🏆 Tournament listings and leaderboards
- 🔍 Course search and filtering

## Running Locally

```bash
npm install
npm run dev
```
